# OSOS - Book Store - Front End

## Set Up
### Clone project
`git clone https://github.com/deshandxdy/OSOS-frontend.git`

### packages Install (Tailwind css. Flowbite, etc. )
`npm install`

### Set backend api URL
Open src/app/constants.ts file and replace backend api url

## Run Project
`ng serve`
